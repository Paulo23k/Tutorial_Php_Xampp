Entre no CMD ou POWERSHELL


SEGUE OS COMANDO ABAIXO

Use "cd" e o caminho da pasta até o php, No meu caso é cd C:\php

Em seguida use o comando (php -S localhost:80 -t "CAMINHO COMPLETO ATÉ A PASTA ONDE ESTARÁ O ARQUIVO EM php") e enter para ativar o server
